Setup guide.

1. Get your Gemini API key (free)

  1. Go to aistudio.google.com/apikey
  2. Sign in with Google → Create API key → Copy it
  3. Paste it in Settings above and hit Save

2. Set up Supabase (free tier, for session history)

  1. Go to supabase.com → New project
  2. Settings → API → copy Project URL and anon public key
  3. Go to SQL Editor and run:
    CREATE TABLE sessions (id uuid DEFAULT gen_random_uuid() PRIMARY KEY, user_id text, type text, title text, content jsonb, created_at timestamptz DEFAULT now());
  4. Paste URL and key in Settings above

3. Enable Google Login (requires deployment)

  1. Deploy this file to Vercel or GitHub Pages
  2. In Supabase → Authentication → Providers → enable Google
  3. Add your deployed URL as redirect URL in both Supabase and Google Console
  4. Sign in with Google button will then work from your deployed URL

4. Local use (no deployment needed)

Everything except Google OAuth works on file:// — just open this HTML file directly. Gemini intel fetch, all prompt generators, session storage (localStorage fallback) all work offline-ready.
